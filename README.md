# About Me

A personal portfolio website showcasing my background, projects, and skills.
The site is built with plain HTML, CSS, and a few images, and is designed to be lightweight and responsive.

## Features

- **Responsive layout** – adapts to mobile, tablet, and desktop screens.
- **Clean design** – minimalistic styling with a focus on readability.
- **Project gallery** – displays a selection of projects with photos and descriptions.

## How to View

Open `index.html` in a web browser. No build steps are required.
